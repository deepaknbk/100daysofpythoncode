 **Boiler Plate template for Decorators**

_More details on decorators can be found in Day22 folder_  

    import functools
    def decorator(func): 
    @functools.wraps(func)
    def wrapper_decorator(*args, **kwargs):
        # Do something before
        value = func(*args, **kwargs)
        # Do something after
        return value
    return wrapper_decorator     
    
     def name(arg_1, ...):
